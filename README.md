# Aston Intensive

Проект прохождения курса "Aston Intensive" состоит из 8 домашних заданий на разные темы по Java
Backend Development. В основе заданий лежит проект `User Service` (`t02-user-service`),
предоставляющий базовый учет пользователей в системе. Каждое следующее домашнее задание:
`t03-testing`, `t04-spring` и так далее - является развитием базового проекта и практическим
применением необходимых технологий backend'а.
